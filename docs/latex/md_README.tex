Yo\+Yo-\/\+S\+LA is a python module providing required functionalities to view/edit/modify labeled data using photo editing softwares like G\+I\+MP. The library supports various file formats such as xcf (gimp) and ora (Open\+Raster).\hypertarget{md_README_autotoc_md1}{}\doxysection{Features}\label{md_README_autotoc_md1}

\begin{DoxyItemize}
\item A customized version of Open\+Raster file ($\ast$.orax).
\item A File loader for multi-\/layer imagery files (the loader load the file to a in-\/memory structure containing the layers, original image, metadata, and other properties)
\item Extension of the file loader for X\+CF files ($\ast$.xcf)
\item Extension of the file loader for O\+RA files ($\ast$.ora)
\item Extension of the file loader for O\+R\+AX files ($\ast$.orax)
\item Extension of the file loader for R\+LE files.
\item A File writer for multi-\/layer imagery files (updating the file structure based on the modifications made)
\item The in-\/memory structure should have lazy loading of the layers.
\item The in-\/memory structure should provide access to the metadata using \mbox{[}string index\mbox{]}.
\item A Sequential loader for processing files inside a folder.
\item Convert from a multi-\/layer format to another multi-\/layer format.
\item A tool for opening the O\+R\+AX files in gimp (including the gimp plugin, linux file handler, and script to install the file handler).
\item A tool for converting all files in a folder from a file format to another file format.
